# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-07

### Added
- Initial project structure with multi-language support
- Python package with code engine and hotkey manager
- TypeScript overlay manager implementation
- Rust core engine with configuration support
- Comprehensive build system using Make
- Docker support for containerized deployment
- CI/CD pipeline with GitHub Actions
- Cross-platform build support (Windows, macOS, Linux)
- Testing infrastructure for all languages
- Linting and formatting configuration
- Packaging and distribution scripts
- Comprehensive documentation

### Components
- Python backend for core functionality
- TypeScript for UI and overlay features
- Rust for high-performance engine components
- Docker image for easy deployment

### Build System
- Makefile with all common tasks
- Automated dependency management
- Multi-platform artifact generation
- Build verification scripts

### CI/CD
- Automated testing on multiple platforms
- Code quality checks
- Docker image building and publishing
- Release automation
